# Cier-sur

CMS para el cier-sur
### Requerimientos
* Python 3
* Django 1.8.2
* django-markdown
* weasyprint

### Instalación

```sh
$ git clone https://github.com/overjt/ciersur.git
$ cd ciersur
$ python manage.py runserver
```
